# 🧬 THE PURPLE STACK

**Human OS v5.0 — System Architecture for the Mad King**

This repository documents a modular, versioned framework for tracking cognitive performance, recovery, sleep, and supplement experiments.

This is not a random supplement list. Every component has a job. Every module has KPIs. Every change is versioned. Every outcome is measured.

**This is engineering, not guessing.**

## 🎯 Goals

- Optimise cognitive performance for AuDHD
- Reduce nervous-system hyperarousal
- Improve sleep initiation and maintenance
- Track outcomes with telemetry
- Iterate like software

## 📊 Method

1. Establish a baseline.
2. Change one controlled variable where practical.
3. Track sleep, focus, mood, RLS, migraine, and recovery metrics.
4. Review results over a defined period.
5. Keep, revise, or remove the change.
6. Record the decision in the changelog.
